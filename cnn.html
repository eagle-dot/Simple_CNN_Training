Neural Network Backpropagation Training Walkthrough

This document outlines a complete, step-by-step mathematical example of the backpropagation algorithm using a minimal Multi-Layer Perceptron (MLP) architecture.

1. Network Architecture & Configuration

The network consists of 4 total neurons: 2 in a single hidden layer and 2 in the output layer.

  • Inputs: x1 = 0.05, x2 = 0.10
  • Hidden Layer: Neurons h1, h2 (with biases b1 = 0.35)
  • Output Layer: Neurons o1, o2 (with biases b2 = 0.60)
  • Targets (Desired Outputs): t1 = 0.01, t2 = 0.99
  • Activation Function: Sigmoid, σ(z) = 1 / (1 + e-z)
  • Learning Rate: η = 0.5

Initial Weight Allocations

  • Input-to-Hidden Weights: w1 = 0.15, w2 = 0.20, w3 = 0.25, w4 = 0.30
  • Hidden-to-Output Weights: w5 = 0.40, w6 = 0.45, w7 = 0.50, w8 = 0.55

2. The Forward Pass

To begin training, we calculate the total net input to each hidden layer neuron, apply the activation function, and pass the results to the output layer.

Hidden Layer Calculations

neth1 = w1x1 + w2x2 + b1 = (0.15)(0.05) + (0.20)(0.10) + 0.35 = 0.3775
outh1 = σ(neth1) = 1 / (1 + e-0.3775) = 0.593265
neth2 = w3x1 + w4x2 + b1 = (0.25)(0.05) + (0.30)(0.10) + 0.35 = 0.3925
outh2 = σ(neth2) = 1 / (1 + e-0.3925) = 0.596884

Output Layer Calculations

Using the outputs of the hidden layer as the new inputs:

neto1 = w5outh1 + w6outh2 + b2 = (0.40)(0.593265) + (0.45)(0.596884) + 0.60 = 1.105904
outo1 = σ(neto1) = 1 / (1 + e-1.105904) = 0.751365
neto2 = w7outh1 + w8outh2 + b2 = (0.50)(0.593265) + (0.55)(0.596884) + 0.60 = 1.224919
outo2 = σ(neto2) = 1 / (1 + e-1.224919) = 0.772928

Total Error Calculation (Squared Error)

Etotal = ∑ ½ (target - output)2
Eo1 = ½ (0.01 - 0.751365)2 = 0.274811
Eo2 = ½ (0.99 - 0.772928)2 = 0.023560
Etotal = 0.274811 + 0.023560 = 0.298371

3. The Backward Pass (Backpropagation)

The goal of backpropagation is to find the partial derivative of the total error with respect to each weight, allowing us to update the model parameters.

Phase 1: Updating Hidden-to-Output Weights (Example: w5)

Applying the Chain Rule:

∂Etotal / ∂w5 = (∂Etotal / ∂outo1) × (∂outo1 / ∂neto1) × (∂neto1 / ∂w5)
  • Error gradient with respect to output: ∂Etotal / ∂outo1 = -(target1 - outo1) = -(0.01 - 0.751365) = 0.741365
  • Output gradient with respect to net input: ∂outo1 / ∂neto1 = outo1(1 - outo1) = 0.751365(1 - 0.751365) = 0.186816
  • Net input gradient with respect to weight: ∂neto1 / ∂w5 = outh1 = 0.593265

Multiplying the components together:

∂Etotal / ∂w5 = 0.741365 × 0.186816 × 0.593265 = 0.082167

Weight Update for w5:

w5new = w5 - η(∂Etotal / ∂w5) = 0.40 - (0.5)(0.082167) = 0.358916

Phase 2: Updating Input-to-Hidden Weights (Example: w1)

∂Etotal / ∂w1 = (∂Etotal / ∂outh1) × (∂outh1 / ∂neth1) × (∂neth1 / ∂w1)

First, pooling the downstream output layer errors:

∂Etotal / ∂outh1 = [(∂Eo1 / ∂neto1) × w5] + [(∂Eo2 / ∂neto2) × w7] = 0.055399 + (-0.019030) = 0.036369

Next, completing the chain rule components for w1:

  • ∂outh1 / ∂neth1 = 0.593265(1 - 0.593265) = 0.241301
  • ∂neth1 / ∂w1 = x1 = 0.05
∂Etotal / ∂w1 = 0.036369 × 0.241301 × 0.05 = 0.0004388

Weight Update for w1:

w1new = w1 - η(∂Etotal / ∂w1) = 0.15 - (0.5)(0.0004388) = 0.149780

4. Summary of Parameter Adjustments

Weight Parameter Initial Value Value After 1 Epoch
w1 (Input Layer → Hidden Layer) 0.15 0.149780
w5 (Hidden Layer → Output Layer) 0.40 0.358916

5. Architectural Clarification

  • Layer Count: This is structurally classified as a 2-layer neural network because it features two distinct layers containing trainable parameters (the hidden layer and the output layer). The input layer contains no weights and is omitted from the formal layer count.
  • CNN Distinctions: While this network maps inputs to predictions, it is a standard Fully Connected Network (MLP) and not a Convolutional Neural Network (CNN). Converting this design into a true CNN would require structural additions such as Convolutional Layers (applying specialized kernels to capture spatial/temporal features) and Pooling Layers (such as Max Pooling) to downsample features before passing data into fully connected output nodes.
